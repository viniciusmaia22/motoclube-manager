# Motoclube Manager

Sistema de gerenciamento de motoclube.

## Objetivo

O objetivo do projeto é criar uma aplicação web com React e TypeScript no frontend e ASP.NET Core Web API em C# no backend.

O sistema será desenvolvido de forma incremental, começando pelo cadastro de membros do motoclube.

## Stack inicial

- React
- TypeScript
- CSS puro
- ASP.NET Core Web API
- C#
- API em memória na primeira fase
- Git e GitHub

## Primeiro módulo

O primeiro módulo será o Cadastro de Membros.

Funcionalidades previstas:

- listar membros;
- cadastrar membro;
- editar membro;
- excluir membro;
- visualizar detalhes do membro;
- filtrar membros por status.
